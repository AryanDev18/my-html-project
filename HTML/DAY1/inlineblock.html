inline vs block

This is an example of inline and block elements in HTML.

The inline elements do not start on a new line and only take up as much width as necessary. For example, the <b> tag is an inline element.

On the other hand, block elements start on a new line and take up the full width available. For example, the <h1> tag is a block element.
